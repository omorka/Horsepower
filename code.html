                            //get the values from startValue and endValue
                            function getValues()
                            {
                                let startValue = document.getElementById("startValue").value;
                                let endValue = document.getElementById("endValue").value;

                                //parse into Integers
                                startValue = parseInt(startValue);
                                endValue = parseInt(endValue);

                                if (Number.isInteger(startValue) && Number.isInteger(endValue)) 
                                {  
                                    //generate numbers
                                    let numbers = generateNumbers(startValue, endValue);
                                    //display numbers
                                    displayNumbers(numbers);
                                }
                                else
                                {
                                    //send error message
                                    alert("Please enter integers!");
                                }

                            }
GetValues() Method

The getValues() method gets the input from the HTML and stores it in javascript values that are then checked to confirm they are integers before being passed into the next method.

                        
                            //generate numbers from startValue to the endValue
                            function generateNumbers(startValue, endValue)
                            {
                                let numbers = [];

                                for(let i = startValue; i <= endValue; i++)
                                {
                                    numbers.push(i);
                                }

                                return numbers;
                            }
GenerateNumbers() Method

The generateNumbers() method takes in 2 parameters which are the startValue and endValue from the getValues() method and pushes the numbers from the starting value to the ending value in an array using a FOR Loop.

                        
                            //display the numbers and mark even numbers bold
                            function displayNumbers(numbers)
                            {
                                let templateRows = "";

                                for(let i = 0; i < numbers.length; i++)
                                {
                                    let number = numbers[i];

                                    if(number % 2 == 0)
                                    {
                                        templateRows += `<tr><td><strong>${number}</strong>
                                        </td></tr>`
                                    }
                                    else
                                    {
                                        templateRows += `<tr><td>${number}</td></tr>`
                                    }
                                }

                                document.getElementById("results").innerHTML = templateRows;
                            }
DisplayNumbers() Method

The displayNumbers() method takes in the array from the generateNumbers method and using a FOR loop iterates over the numbers array and adds the templatRows string to the HTML document that will then display them in a table.
